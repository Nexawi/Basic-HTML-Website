# Basic HTML Website

A multi-page personal website built with semantic HTML — no CSS frameworks or styling, focusing purely on HTML structure and layout.

## Pages

- **Home** (`index.html`) — Landing page with navigation and intro
- **Projects** (`Projects.html`) — Showcase of projects
- **Articles** (`Articles.html`) — Articles/blog section
- **Contact** (`Contact.html`) — Contact information

## Features

- Semantic HTML structure
- SEO meta tags
- Open Graph (OG) tags
- Favicon and web manifest
- Fixed navigation bar across pages
